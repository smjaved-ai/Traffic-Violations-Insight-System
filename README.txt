**********************************************************
Traffic Violations Insight System
**********************************************************

###Description###
	Urban safety agencies generate millions of traffic-violation records each year, capturing details such as violation type, location, driver demographics, vehicle information, and enforcement-related metadata. However, this raw data is often noisy, inconsistent, duplicated, and difficult to interpret without systematic preprocessing and visualization
	This project demonstrates complete data analytics system that transforms a large, raw dataset of traffic violations (≈ 10 lakh rows) into actionable insights through Exploratory Data Analysis (EDA), data cleaning, preprocessing, and an interactive visualization dashboard built with Streamlit.

**********************************************************

###Pre-Requisites####
  Latest google chrome or Microsoft Edge Browser
  Python (3.8 or later recommended)
  Streamlit
  SQL database (MySQL Workbench)
